# HotMap
# 热力图生成器

演示地址：[www.skyzgh.com](http://www.skyzgh.com)

github部署地址：[https://skyzgh-cn.github.io/HotMap/](https://skyzgh-cn.github.io/HotMap/)  

#### 介绍
通过百度地图实现热力图生成

#### 软件架构
百度地图JS API实现功能


#### 安装教程

HTML无需部署，下载就可以体验效果

#### 使用说明

1.  左侧文本框输入地址，每行输入一个
2.  第二步：点击批量标记
3.  等待标记结束点击显示热力图
